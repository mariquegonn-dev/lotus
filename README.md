# ✍🏻 Como instalar

Clone o projeto, utilizando o terminal:

```bash
  git clone https://github.com/mariquegonn-dev/lotus
```

Em seguida:

```bash
  npm install
```

Para rodar o projeto:

```bash
  npm run dev
```
